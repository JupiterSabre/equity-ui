# Icon

Icon component with different props

#### Example

<Canvas of={Icon.Basic} />

## Usage

```ts
import { Icon } from "@femmecubator/equity-ui";

const Example = () => {
  return (
      <Icon name="email" />
  );
};

export default Example;
```

#### Arguments

- **name** - Specifies the name of the icon to be rendered.
- **size** - Specifies the size of the icon to be rendered. Could be one of the following: `small`, `large`, or a numeric value in pixels.
